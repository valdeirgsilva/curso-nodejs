Como esta tarefa foi construída?

  1. Foi feita com node no backend em conjunto com o framework Express;
  2. As rotas foram feitas com express.Router;
  3. Para servir os assets foi utilizado express.static;
  4. O servidor foi configurado para receber conexões na porta 5000;
  5. Por fim, nodemon foi adicionado como depndência de desenvolvimento para aumentar a produtividade.
